# How to Open a Code Along Notebook

Welcome! This page explains how to open a code along notebook in Google Colaboratory (Colab) during a live session or on your own.

## Important Note

In these instructions, replace `<yourUserNameHere>` with your exact GitHub username whenever you see it.

## How to Open a Code Along Notebook in Google Colab

1. You have already **accepted the code along invitation from Canvas**. When you did that, GitHub Classroom created your personal repo (e.g., `yse-eds-cert/code_along_notebook-<yourUserNameHere>`).

2. After accepting, **go to your personal repo** on GitHub:
   `https://github.com/yse-eds-cert/code_along_notebook-<yourUserNameHere>`
   *(Replace `<yourUserNameHere>` with your personal GitHub username.)*

3. **Open the notebook in Google Colab.** Your repo contains one or more `.ipynb` notebook files. To launch a notebook:

   - **Option A** — Edit the following link with your GitHub username and the notebook filename, then follow the link:
     `https://colab.research.google.com/github/yse-eds-cert/code_along_notebook-<yourUserNameHere>/blob/main/<notebook_name>.ipynb`
     *(Replace `<yourUserNameHere>` with your username and `<notebook_name>` with the name of the notebook file.)*

   - **Option B** — Open Google Colab directly:
     1. Go to https://colab.research.google.com/
     2. Click the **GitHub** tab in the pop-up
     3. For the repository, choose `yse-eds-cert/code_along_notebook-<yourUserNameHere>` (replacing `<yourUserNameHere>` with your username)
     4. Click the notebook file (`.ipynb`) you want to open

4. In Colab, go to **File → Save a copy in Drive** — this creates an autosaving working copy in your Google Drive. Do all your work there so you always have a copy backed up. You may delete "Copy of" from the file name.

5. Follow along with the instructor during the live session, or work through the notebook on your own.

6. When finished, save your work back to GitHub by going **File → Save a copy in GitHub** → select your personal repo (`yse-eds-cert/code_along_notebook-<yourUserNameHere>`). **Be sure to save to *YOUR* personal repo by correctly replacing `<yourUserNameHere>`.**

> **Why save to Drive first?** Colab autosaves to Google Drive but does **not** autosave to GitHub. Saving to Drive protects your work-in-progress. Saving to GitHub is the manual submission step.
