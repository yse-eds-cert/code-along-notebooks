# How to Open a Code Along Notebook

This page explains how to open a code along notebook in Google Colaboratory (Colab) during a live session or on your own.

## Orienting to where you are

If you're reading this README, you've **already accepted the code along assignment** through the Classroom50 link in Canvas. When you accepted, Classroom50 automatically created your own personal copy of this repository — named `yse-eds-cert-code-along-notebooks-<yourUserNameHere>` — which is where you are right now.

> **About the placeholders:** Replace `<yourUserNameHere>` with your exact GitHub username whenever you see it. Replace `<file_path_and_notebook_name>` with the file you would like to open, including the folders that lead to it and the `.ipynb` at the end.

## How to Open a Code Along Notebook in Google Colab

1. **Confirm you're in your personal repo.** Your browser should be at `https://github.com/yse-eds-cert/yse-eds-cert-code-along-notebooks-<yourUserNameHere>` (*`<yourUserNameHere>` is your personal GitHub username*). Your repo contains many `.ipynb` notebook files, organized into folders that match the certificate program's courses and modules.

2. **Open the notebook in Google Colab.** To launch a notebook:

   - **Option A (Recommended)** — Open Google Colab directly:
     1. Go to https://colab.research.google.com/
     2. Click the **GitHub** tab in the pop-up
     3. For the repository, choose `yse-eds-cert/yse-eds-cert-code-along-notebooks-<yourUserNameHere>` (replacing `<yourUserNameHere>` with your username)
     4. Click the notebook file (`.ipynb`) you want to open

   - **Option B** — Edit the following link with your GitHub username and the notebook filename, then follow the link:
     `https://colab.research.google.com/github/yse-eds-cert/yse-eds-cert-code-along-notebooks-<yourUserNameHere>/blob/main/<file_path_and_notebook_name>.ipynb`
     *(Replace `<yourUserNameHere>` with your username and `<file_path_and_notebook_name>` with the path to the notebook file.)*
      - You can find the notebook you need both in Canvas and by clicking into the course you're in, the module of that course, and finding the notebook associated with the lecture you're on.

3. Follow along with the instructor during the live session, or work through the notebook on your own.
4. Note: You don't have to "Save a copy in GitHub" unless you made edits to the notebooks that you'd like to save in your personal GitHub repo. These are not graded, and are meant only as an instructive tool.
